This is a Next.js Project

This file is a placeholder. Your portfolio is a dynamic application that needs to be run with a server or built into static files.

For Development

To view your portfolio with live updates as you make changes:

  1. Install dependencies: npm install
  2. Start the dev server: npm run dev

Then, open http://localhost:3000 in your browser.

For Deployment (Static HTML)

I have configured the project to export a static HTML version. You can host these files on any static web host.

  1. Build the static site: npm run build

After the build is complete, you will find all the static files, including index.html, in a new folder named /out.

You can then upload the contents of the /out folder to your web host.
